## 🚀 Quick Start

### Prerequisites
- Node.js 16+ 
- npm or yarn
- Base44 account (for backend services)

### Installation

```bash
# Clone repository
git clone https://github.com/nikolai2244/MWSportsMasters2026.git
cd MWSportsMasters2026

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Edit .env with your Base44 credentials

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
